## [0.2.0] - 2025-11-01

### Added

* Text-driven animation pipeline for TextMesh Pro targets with modular Transform, Color, and Material subsystems.
* Comprehensive tag library (`{wave}`, `{rainbow}`, `{charByChar}`, etc.) with reusable parameter definition assets.
* Custom UI Toolkit inspectors leveraging FLFloppa Editor Helpers for all ScriptableObject assets.
* `TextAnimator` MonoBehaviour facade with runtime playback API (`Play`, `Stop`, `Reinitialize`).
* Documentation set: README, Editor Setup Guide, Tag Reference, use-case walkthroughs.
* Sample scaffolding (`Samples~/BasicSetup`) to demonstrate registry, parameter, and subsystem wiring.
